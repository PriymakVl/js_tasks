Shulte Table - JavaScript Task

Задание

  1. Написать скрипт который создает таблицу Шульте
  2. в таблице 25 ячеек
  3. в каждой ячейке при загрузке генерируется случайное число от 1 до 25
  4. таблица занимает весь экран
  5. В правом верхнем углу создать кнопку для обновления таблицы
  6. Появляется только при наведении курсора на эту область экрана
