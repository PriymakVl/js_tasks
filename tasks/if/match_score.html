Match Score - JavaScript Task

Задание

  1. При клике по кнопке игрока увеличивать его счета в игре на 1
  2. Прекращать игру при достижении победного счета в игре указанного на экране
  3. Показать строку сообщающую какой игрок стал победителем
  4. Пометь цвет счета победителя на желтый
  5. Начать игру снова при клике на кнопку "начать снова"
