Add Border Input - JavaScript Task

Задание

  1. При потере фокуса, проверить содержимое input на правильное количество введенных символов.
  2. Нужное количество символов указывается в атрибуте data-length
  3. Если введено правильное количество символов, то рамка вокруг инпута меняет цвет на зеленый, если неправильно то на красный
