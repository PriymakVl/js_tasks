Text Change Colort - JavaScript Task

Задание

  1. Написать скрипт меняющий цвет текста
  2. Скрипт запускается после нажания на кнопку «Запустить скрипт»
  3. Цвет текста меняется через 2 секунды и производится по кругу
  4. После нажатия надпись на кнопке меняется на «Остановить скрипт»
  5. После нажатия на кнопку «Остановить скрипт» смена цвета останавливается и цвет текста становится черным
