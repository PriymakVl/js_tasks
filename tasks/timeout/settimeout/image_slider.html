Image Slider - JavaScript Task

Задание

  1. Написать скрипт слайдера
  2. По нажатию на кнопку «start» запускается работа слайдера
  3. По нажатию на кнопку «stop» работа слайдера останавливается
  4. Переключение изображений происходит автоматически, через заданный промежуток времени
