Fly Catch - JavaScript Task

Задание

  1. Написать скрипт игры «Охота на муху!»
  2. Муха прявляется в случайном месте экрана
  3. Пропадает при клике по ней
  4. Пропадает через 2 секунду если по ней не попали
  5. Игра заканчивается через 30 секун
  6. Игроку показывается количество попаданий
